HTML

  • The head element contains information are not visible
  • The body element contains the content that is shown to the user
  • !DOCTYPE html tells the browser its an HTML page
  • html opening tag, which represents the root
  • lang="en" tells the browser regarding the language
  • The style element defines the CSS styles associated with a website
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • h1 is the opening tag and /h1 is the closing tag
  • The h1, h2, h3, h4, h5, and h6 elements represent the level of heading
  • The p element represents a paragraph or block of text
  • The ul, ol, and li elements represent unordered lists
  • img element contains information about images that are displayed on the webpage
  • src attribute defines the location of the image
  • br element creates a line of empty space
  • a creates links to webpage

CSS

  • Is a programming language used by developers to define how webpage content is presented to users.
  • style attribute to change the color of a single h1 heading
  • margin property acts much like a margin in a word processing documen
  • margin property indicates how much space we want around the outside of an element
  • he padding property adds space around the content inside an element
